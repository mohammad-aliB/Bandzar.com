Introduction To Factorials(n!)

By:Mohammad-Ali Bandzar | July 2 2019

The factorial function in math written as n! where n can be any whole number greater than or equal to zero. is a fancy way of telling us to take the product(multiply) all whole numbers between 1 and itself together.(with zero being a special case we will discuss below)

    <p id="output"></p>
    <script>
        function findFactorial() {
            var x = document.getElementById("inputFactorial").value;
            if(x%1 != 0 || x<0){
                document.getElementById("output").innerHTML = "please enter a valid input";
            }else if(x==0){
                document.getElementById("output").innerHTML = "0";
            }else{
                var n=1;
                while(x>0){
                    n=n*x;
                    x=x-1;
                }
                document.getElementById("output").innerHTML = n;

            }
        }
    </script>
